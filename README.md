# CRUD em PHP com DataTables

Projeto desenvolvido como desafio do estágio, com o objetivo de criar um CRUD utilizando PHP e a biblioteca DataTables para exibição e manipulação dos dados.

## Funcionalidades

- Listagem de registros
- Cadastro de novos dados
- Edição de registros
- Exclusão de registros
- Paginação, busca e ordenação com DataTables

## Tecnologias

- PHP
- MySQL
- HTML
- CSS  
- DataTables
